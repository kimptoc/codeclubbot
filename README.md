# 🤖 Kilo Chat

A simple web app that lets you chat with Kilo's AI models.

## Usage

1. Open https://cattyclaw-bot.github.io/codeclubbot/ in a web browser
2. Enter your Kilo API key from [kilo.ai](https://kilo.ai)
3. Type a prompt and click Send

## Tech

- Plain HTML/JS (no build needed)
- Uses `@kilocode/sdk` from esm.sh
- Runs entirely in the browser
